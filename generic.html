Return on Investment of College Football Programs

DATA COLLECTION ● DATA CLEANING ● MULTIPLE LINEAR REGRESSION ● DATA ANALYSIS ● TABLEAU ● EXCEL

S u m m a r y

This is a semester-long project I pursued in my career-based writing and rhetoric class because I wanted to practice answering a complex question with data science and being able to present it to an audience unfamiliar with statistics or data science. This is similar to something I see myself doing in my future career, and it is a practiced skill to be able to break down technical data findings into easily interpretable information. I collected college football financial and standing data to create my own regression in order to determine how a program's revenues and expenses can affect its standing and performance.
